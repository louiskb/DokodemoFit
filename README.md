# Dokodemo Fit
AI tool that creates workout routines and meal plans depending on goals and equipment available

## Technology Stack

The app was built using Ruby on Rails, and uses the Ruby-LLM Gem to call to the OpenAI LLM provided by GitHub. The number of tokens is limited, so the tool will not work when under load.

## Disclaimer

The advice provided by Dokodemo Fit is generated by an LLM, and therefore may be inappropriate or contain hallucinations. Always use your own judgement when getting recommendations from AI.
